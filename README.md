# Data_analysis_over_customer_shopping
Customer shopping behavior analysis using SQL, Python, and Power BI to uncover purchasing patterns, customer insights, and business-driven visualizations.
## Project Overview
This project analyzes customer shopping behavior using SQL, Python, and Power BI to uncover purchasing patterns, customer insights, and business trends.

The workflow includes:
- Data cleaning and preprocessing using Python
- SQL queries for business analysis
- Interactive dashboard creation in Power BI
- Exploratory Data Analysis (EDA)

---

## Objectives
- Analyze customer purchasing behavior
- Identify revenue and sales trends
- Understand discount impact on purchases
- Discover customer segmentation insights
- Build an interactive business dashboard

---

## Dataset
Dataset used:
`customer_shopping_behavior.csv`

Features analyzed include:
- Customer demographics
- Purchase amount
- Item purchased
- Discount applied
- Payment method
- Frequency of purchase
- Category preferences
